BIG PROJECT
Xylen Project
Xylen Project merupakan bot Telegram berbasis Python yang dikembangkan sendiri (tanpa tim). Bot ini dirancang untuk keperluan OSINT, pentesting, dan analisis digital lain pada ranah yang diperbolehkan. Fitur utama meliputi:
  • OSINT: mencari data berdasarkan satu informasi (mis. nama) dan mencocokkan ke database untuk menemukan jejak terkait.
  • FileInt: menganalisis file untuk melacak metadata atau jejak—misalnya mendeteksi indikasi lokasi pada foto sehingga user dapat menelusuri informasi lebih lanjut terkait foto tersebut.
  • IQC: pembuatan iPhone quote chat (format quote khusus untuk keperluan presentasi atau dokumentasi).
  • QRGen: pembuatan QR yang berisi pesan atau mengarah ke URL; QR dapat dimodifikasi (warna, background transparan).
  • Kodepos Checker: mencari lokasi berdasarkan kodepos di database berisi informasi kode pos di seluruh negara.
  • PhoneInt: pencarian ISP dan lokasi kasar; menggunakan database ISP global untuk akurasi yang lebih baik, intinya ini menggunakan database berisi daftar isp seluruh dunia, kecil maupun besar.
  • URL Checker: analisis mendalam sebuah URL untuk mencari informasi seperti IP asli, versi, atau indikator teknis lain, dan informasi lainya secara deep.
  • AutBugHunt: modul yang bertujuan men-scan potensi celah menggunakan curl secara otomatis sebagai proof-of-concept internal. fitur ini benar benar menggunakan seluruh command curl, full automasi, user hanya perlu memasuki url
masih banyak lagi fitur fitur lainya, namun masih tahap proses pengembangan — penggunaan untuk seminggu gratis, dimohon untuk melaporkan bug atau request fitur yang tidak biasa untuk mempermudah cybersecurity, username bot @YourXylenBot.
