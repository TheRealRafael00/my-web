Big Project — ovtb project

ovtb project merupakan bot Telegram berbasis Python yang dikembangkan sendiri (tanpa tim). Bot ini dirancang untuk keperluan OSINT, pentesting, dan analisis digital lain pada ranah yang diperbolehkan.

Fitur utama meliputi:

  • OSINT: mencari data berdasarkan satu informasi (mis. nama) dan mencocokkan ke database untuk menemukan jejak terkait.
  • FileInt: menganalisis file untuk melacak metadata atau jejak—misalnya mendeteksi indikasi lokasi pada foto sehingga user dapat menelusuri informasi lebih lanjut terkait foto tersebut.
  • IQC: pembuatan iPhone quote chat (format quote khusus untuk keperluan presentasi atau dokumentasi).
  • QRGen: pembuatan QR yang berisi pesan atau mengarah ke URL; QR dapat dimodifikasi (warna, background transparan).
  • Kodepos Checker: mencari lokasi berdasarkan kodepos di database berisi informasi kode pos di seluruh negara.
  • PhoneInt: pencarian ISP dan lokasi kasar; menggunakan database ISP global untuk akurasi yang lebih baik.
  • URL Checker: analisis mendalam sebuah URL untuk mencari informasi seperti IP asli, versi, atau indikator teknis lain secara deep.
  • AutBugHunt: modul yang men-scan potensi celah menggunakan curl secara otomatis, full automasi — user hanya perlu memasukkan URL.

Masih banyak lagi fitur-fitur lainya, namun masih tahap pengembangan — penggunaan gratis 1 minggu. Laporkan bug atau request fitur yang tidak biasa untuk membantu keamanan siber.
Username bot: @ovtlsbot

ovtb project is a Python-based Telegram bot developed independently (no team). Designed for OSINT, pentesting, and other digital analysis within permitted scope.

Main features include:

  • OSINT: search data based on a single clue (e.g., name) and match it with databases to find related traces.
  • FileInt: analyze file metadata — e.g., detect geolocation info inside photos for deeper research.
  • IQC: generate iPhone-style quote chats for documentation or presentation.
  • QRGen: create QR codes containing messages or URLs; fully customizable color & background.
  • Postal Code Checker: find locations using postal code databases from all regions.
  • PhoneInt: lookup ISP and approximate location using a global ISP database for accuracy.
  • URL Checker: perform deep analysis of a URL to reveal original IP, version, and other indicators.
  • AutBugHunt: auto-scan vulnerabilities using full curl commands — user just inputs the URL.

More features are still in development — free for 1 week of usage. Please report bugs or unique feature requests to help improve cybersecurity.
