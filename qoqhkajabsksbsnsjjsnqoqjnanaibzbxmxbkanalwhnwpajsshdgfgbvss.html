⚡ Saatnya Kita Praktekkan Nmap

Sekarang saatnya Anda mempraktikkan penggunaan Nmap. Sebelum menjalankan tool ini, tentukan terlebih dahulu website atau host yang ingin Anda analisis. Saran untuk latihan awal, gunakan target yang legal dan aman, misalnya: scanme.nmap.org (host resmi yang memang disediakan oleh pengembang Nmap untuk belajar).

Nmap merupakan salah satu tool terpenting dalam dunia pentesting untuk melakukan network scanning dan enumerasi port. Tool ini dapat dijalankan baik di Termux maupun di Kali Linux.

Untuk memudahkan pembelajaran, di bawah ini telah disiapkan sebuah script Python sederhana yang bisa Anda gunakan secara gratis. Script ini akan membantu Anda memahami cara dasar menggunakan Nmap

Cara mendapatkan script ini: tekan tombol download di bawah. Script akan otomatis menyalin perintah wget yang bisa langsung Anda jalankan di Termux atau Kali Linux.

Setelah file berhasil diunduh, jalankan dengan perintah:
python nmap.py
